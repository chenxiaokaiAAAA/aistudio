# 阿里云服务器-最完整版本部署流程

## 需要安装的依赖

### 系统依赖
- Python 3.x
- PostgreSQL
- Redis
- Nginx

### Python依赖
```bash
pip install -r requirements.txt
```

## 需要运行的脚本

1. 数据库初始化：`python scripts/database/setup_postgresql.py`
2. 数据迁移：`python migrate_data_auto.py`
3. 启动服务：`python start_production.py`

详细说明请参考：`docs/deployment/生产环境部署指南.md`
