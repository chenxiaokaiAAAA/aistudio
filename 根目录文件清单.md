# 根目录文件清单

## ✅ 必需保留的文件（核心运行文件）

### 主应用文件
- `test_server.py` - Flask主应用文件（必需）
- `start.py` - 开发环境启动脚本（必需）
- `start_production.py` - 生产环境启动脚本（必需）
- `start_with_proxy.bat` - 带代理启动脚本（可选）

### 核心模块
- `order_notification.py` - 订单通知模块（必需）
- `wechat_notification.py` - 微信通知模块（必需）
- `printer_client.py` - 打印系统客户端（必需）
- `local_printer_client.py` - 本地打印客户端（必需）
- `local_printer.py` - 本地打印模块（必需）

### 配置文件
- `gunicorn.conf.py` - Gunicorn配置（必需）
- `requirements.txt` - Python依赖列表（必需）
- `server_config.py` - 服务器配置（必需，但不提交到Git）
- `printer_config.py` - 打印系统配置（必需，但不提交到Git）
- `size_config.py` - 尺寸配置（必需）

### 应用目录
- `app/` - 应用主目录（必需）
- `templates/` - HTML模板（必需）
- `static/` - 静态文件（必需）
- `config/` - 配置文件目录（必需）

## 📄 重要文档（保留在根目录）

- `README.md` - 项目主说明文档（必需）
- `根目录文件说明.md` - 项目文件结构说明（必需）
- `Linux命令快速参考.md` - Linux服务器常用命令速查表（必需）
- `README_重要文档.md` - 重要文档快速索引（必需）

## 📁 其他目录（保留）

- `scripts/` - 脚本目录
- `batch/` - 批处理脚本目录
