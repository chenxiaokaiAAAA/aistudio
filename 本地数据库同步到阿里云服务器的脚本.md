# 本地数据库同步到阿里云服务器的脚本

## 使用方法

### 方法1：使用迁移脚本
```bash
python migrate_data_auto.py
```

### 方法2：使用数据库备份恢复
```bash
# 1. 备份本地数据库
python scripts/database/backup_database.py

# 2. 上传备份文件到服务器
# 3. 在服务器上恢复
python scripts/database/restore_database.py
```
