# 分类导航图标使用说明

## 统一存储路径

分类导航图标统一使用以下路径访问：

- **访问路径**：`/media/category_nav/<文件名>`
- **存储位置**：
  1. `uploads/category_nav/`（优先，通过管理后台上传的图标）
  2. `static/images/category_nav/`（回退，可放置预设图标）

## 使用方法

### 方法一：在管理后台上传图标（推荐）

1. 登录管理后台
2. 进入「首页管理」→「分类导航」→ 点击「分类导航管理」
3. 编辑或创建分类导航项
4. 点击「上传图标」选择图片，上传后会自动保存到 `uploads/category_nav/` 并返回 `/media/category_nav/xxx` 路径

### 方法二：使用预设图标

将图标文件放入 `static/images/category_nav/` 目录，然后在管理界面填写路径：

- `/media/category_nav/群组 1@1x.png`
- `/media/category_nav/群组 2@1x.png`
- 等等

### 已有数据迁移

若数据库中已存在旧路径（如 `/images/category_nav/`、`/images/category-nav/`）：

- 后端 API 会自动转换为 `/media/category_nav/<文件名>`
- 请将对应图标文件拷贝到 `uploads/category_nav/` 或 `static/images/category_nav/`，否则会出现 404

## 注意事项

1. 图标路径使用相对路径（以 `/` 开头），不要使用 `http://` 开头的绝对路径
2. 后端会自动处理旧路径转换，并生成完整 URL
3. 若图标 404，请检查文件是否存在于 `uploads/category_nav/` 或 `static/images/category_nav/`

## 预设图标文件（可选）

若需使用预设图标，可从 `aistudio-小程序/images/category-nav/` 复制到后端 `static/images/category_nav/`，包含例如：

- 群组 1@1x.png、群组 2@1x.png、群组 3@1x.png 等
