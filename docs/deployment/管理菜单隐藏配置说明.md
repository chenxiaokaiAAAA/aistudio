# 管理菜单隐藏配置说明

## 🎯 隐藏目的

根据用户需求，临时隐藏管理后台中的部分菜单选项，以简化界面和管理流程。隐藏的选项保留代码以备将来需要时重新启用。

## ✅ 已隐藏的菜单选项

### 1. 手动新增订单
- **位置**: 管理菜单 → 订单管理 → ❌ 手动新增订单
- **功能**: `/admin/orders/add` - 手动添加订单功能
- **原因**: 简化日常操作，减少意外添加订单的可能性

### 2. 添加商家
- **位置**: 管理菜单 → 签约商家管理 → ❌ 添加商家  
- **功能**: `/admin/add_merchant` - 添加新商家功能
- **原因**: 商家管理暂时不需要频繁添加新商家

### 3. 抖音订单同步
- **位置**: 管理菜单 → ❌ 抖音订单同步
- **功能**: `/admin/douyin` - 抖音平台订单同步功能
- **原因**: 暂时不使用抖音平台功能

## 🛠️ 技术实现

### 隐藏方式
使用HTML注释语法隐藏菜单项，而非删除代码：

```html
<!-- 手动新增订单 - 已隐藏，后续需要时可重新启用 -->
<!-- <li class="list-group-item">
    <a href="/admin/orders/add" class="text-decoration-none">
        <i class="fas fa-plus-circle"></i> 手动新增订单
    </a>
</li> -->
```

- 📁 `templates/admin/dashboard.html` - 主要管理后台页面

## 🔧 恢复方法

### 如何重新启用隐藏的功能

当需要重新启用某个功能时，只需：

1. **打开文件**: `templates/admin/dashboard.html`
2. **找到对应的注释块**
3. **解除HTML注释**: 删除 `<!-- 和 -->` 标记
4. **保存并刷新页面**

### 示例：恢复"手动新增订单"

**隐藏状态**:
```html
<!-- 手动新增订单 - 已隐藏，后续需要时可重新启用 -->
<!-- <li class="list-group-item">
    <a href="/admin/orders/add" class="text-decoration-none">
        <i class="fas fa-plus-circle"></i> 手动新增订单
    </a>
</li> -->
```

**恢复状态**:
```html
<li class="list-group-item">
    <a href="/admin/orders/add" class="text-decoration-none">
        <i class="fas fa-plus-circle"></i> 手动新增订单
    </a>
</li>
```
