# 抖音数据同步配置说明

## 🔧 配置步骤

### 1. 抖音开放平台配置
```python
DOUYIN_CONFIG = {
    'app_key': 'your_app_key',           # 抖音开放平台App Key
    'app_secret': 'your_app_secret',     # 抖音开放平台App Secret
    'base_url': 'https://open.douyin.com',
    'webhook_secret': 'your_webhook_secret'  # Webhook签名密钥
}
```

### 2. 数据库配置（如果使用数据库同步）
```python
DOUYIN_DB_CONFIG = {
    'type': 'mysql',  # 或 'sqlite'
    'host': 'localhost',
    'port': 3306,
    'user': 'douyin_user',
    'password': 'douyin_password',
    'database': 'douyin_shop'
}
```

### 3. Webhook URL配置
在抖音开放平台配置Webhook URL：
```
https://yourdomain.com/api/douyin/webhook
```

## 📊 同步方式对比

| 方式 | 实时性 | 复杂度 | 可靠性 | 推荐度 |
|------|--------|--------|--------|--------|
| API定时同步 | 5分钟延迟 | 中等 | 高 | ⭐⭐⭐⭐ |
| Webhook回调 | 实时 | 高 | 高 | ⭐⭐⭐⭐⭐ |
| 数据库同步 | 1分钟延迟 | 低 | 中等 | ⭐⭐⭐ |

## 🚀 部署步骤

1. **申请抖音开放平台权限**
2. **配置API密钥和Webhook**
3. **集成同步代码到现有系统**
4. **测试同步功能**
5. **配置定时任务或Webhook**

## ⚠️ 注意事项

1. **API限制**: 抖音API有调用频率限制
2. **数据安全**: 确保Webhook签名验证
3. **错误处理**: 实现重试机制
4. **数据一致性**: 避免重复同步
5. **监控告警**: 监控同步状态
