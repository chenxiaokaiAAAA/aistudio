# 删除服务器备份目录

## 🗑️ 删除备份目录命令

在服务器上执行以下命令删除备份目录：

```bash
# 进入根目录
cd /root

# 删除备份目录（逐个删除）
rm -rf project_code_backup2
rm -rf project_code_backup_20260127_100215
rm -rf project_code_backup_20260127_101950

# 或者一次性删除所有匹配的备份目录
rm -rf project_code_backup*
```

## ⚠️ 注意事项

1. **`rm -rf` 命令会永久删除**，无法恢复
2. **确认目录名称**，避免误删重要目录
3. **`project_data` 目录**：如果不确定是否还需要，建议先检查内容

## 🔍 删除前检查

```bash
# 查看目录大小
du -sh /root/project_code_backup*

# 查看目录内容（确认是否真的是备份）
ls -la /root/project_code_backup2
```

## ✅ 完整删除命令（一键执行）

```bash
cd /root && \
rm -rf project_code_backup2 \
       project_code_backup_20260127_100215 \
       project_code_backup_20260127_101950 && \
echo "✅ 备份目录已删除" && \
ls -la | grep project
```

## 📋 关于 project_data 目录

如果不确定 `project_data` 是否还需要，可以先检查：

```bash
# 查看目录内容
ls -la /root/project_data

# 查看目录大小
du -sh /root/project_data

# 如果确认不需要，再删除
rm -rf /root/project_data
```
