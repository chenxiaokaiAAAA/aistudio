# 管理后台配置风格分类绑定操作指南

## 📍 字段位置说明

"绑定风格分类"字段位于**产品编辑表单**中，具体位置如下：

### 1. 添加新产品时

在**添加产品表单**中，"绑定风格分类"字段位于：
- **位置**：在"产品图片"上传区域**下方**
- **顺序**：产品基本信息 → 产品图片 → **绑定风格分类** → 尺寸规格 → 自定义字段

### 2. 编辑现有产品时

在**产品编辑表单**中，"绑定风格分类"字段位于：
- **位置**：在"图片上传"区域**下方**
- **顺序**：产品基本信息 → 图片上传 → **绑定风格分类** → 颜色选项配置说明 → 自定义字段

## 🔍 如何找到编辑表单

### 方法一：点击产品卡片

1. 登录管理后台
2. 进入 **产品管理** → **尺寸管理** (`/admin/sizes`)
3. 在"产品列表"区域，**点击任意产品卡片**
4. 会弹出编辑模态框，在表单中向下滚动即可看到"绑定风格分类"字段

## 📝 配置步骤

### 步骤1：打开产品编辑表单

1. 进入管理后台
2. 导航到：**产品管理** → **尺寸管理**
3. 在"产品列表"中找到要配置的产品
4. **点击产品卡片**（会打开编辑模态框）

### 步骤2：找到"绑定风格分类"字段

在编辑表单中，向下滚动，找到：
- **标题**：`🎨 绑定风格分类`（带有调色板图标）
- **说明文字**：`选择该产品可用的风格分类，加盟商下单时将只显示绑定的风格`

### 步骤3：勾选风格分类

在"绑定风格分类"区域，你会看到所有可用的风格分类，每个分类前面有一个复选框：
- ☐ 拟人风格 👑
- ☐ 油画风格 🎨
- ☐ 其他风格...

**勾选**该产品需要绑定的风格分类。

### 步骤4：保存配置

1. 向下滚动到表单底部
2. 点击 **"保存"** 或 **"更新产品"** 按钮
3. 配置会立即生效

## ⚠️ 注意事项

### 1. 如果没有看到风格分类选项

如果"绑定风格分类"区域显示：
```
暂无风格分类，请先在风格管理中创建
```

**解决方法**：
1. 先进入 **风格管理** 页面
2. 创建风格分类
3. 在风格分类下添加风格图片
4. 返回产品管理页面，即可看到风格分类选项

### 2. 编辑表单是隐藏的

编辑表单默认是隐藏的，需要：
- **点击产品卡片**才会显示
- 或者查看页面源代码，搜索 `绑定风格分类` 找到对应位置

### 3. 字段位置可能因页面版本不同

如果按照上述步骤找不到，可以：
1. 在浏览器中按 `Ctrl+F`（Windows）或 `Cmd+F`（Mac）
2. 搜索关键词：`绑定风格分类`
3. 找到对应的位置

## 💡 提示

如果仍然找不到该字段，可能是：
1. **页面版本不同**：检查 `templates/admin/sizes.html` 文件
2. **权限问题**：确认有产品编辑权限
3. **浏览器缓存**：尝试清除缓存或刷新页面

## 📞 需要帮助？

如果按照以上步骤仍然找不到，请：
1. 检查浏览器控制台是否有错误
2. 查看页面源代码，搜索 `style_category_ids`
3. 确认风格分类是否已创建
