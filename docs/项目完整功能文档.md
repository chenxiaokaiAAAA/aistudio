## 📖 项目概述

### 项目简介

AI拍照机管理系统是一个集成了**微信小程序**、**管理后台**、**加盟商系统**的综合性电商平台，主要用于：

- **AI拍照机服务**：用户上传照片，系统生成艺术作品
- **证件照制作**：支持多种尺寸和底色的证件照制作
- **订单管理**：完整的订单流程管理，从下单到发货
- **加盟商管理**：支持多加盟商、额度管理、订单统计
- **推广分佣**：推广码系统，支持分佣管理

### 核心特点

- ✅ **模块化架构**：代码结构清晰，易于维护和扩展
- ✅ **多端支持**：小程序、管理后台、加盟商系统
- ✅ **完整业务流程**：从下单、支付、制作到发货的完整流程
- ✅ **权限管理**：管理员、运营、加盟商多角色权限
- ✅ **数据统计**：订单统计、业绩分析、报表导出

## 🏗️ 技术架构

### 后端技术栈

- **Web框架**: Flask 2.3.3
- **数据库**: SQLite（开发）/ PostgreSQL（生产可选）
- **ORM**: SQLAlchemy 3.0.5
- **认证**: Flask-Login 0.6.3
- **图片处理**: Pillow 10.0.1
- **二维码生成**: qrcode 7.4.2
- **HTTP请求**: requests 2.31.0
- **Web服务器**: Gunicorn 21.2.0
- **反向代理**: Nginx

### 前端技术栈

- **小程序**: 微信小程序原生开发
- **管理后台**: HTML + CSS + JavaScript（原生）
- **UI框架**: Bootstrap（部分页面）

### 第三方服务

- **微信支付**: 微信小程序支付
- **微信登录**: 微信小程序登录
- **微信订阅消息**: 订单通知
- **AI服务**: 美图API、自定义AI服务商
- **物流系统**: 第三方物流回调

## 🌐 API接口文档

### 小程序API（/api/miniprogram）

- `POST /api/miniprogram/orders` - 提交订单
- `GET /api/miniprogram/orders` - 获取订单列表
- `GET /api/miniprogram/orders/<order_number>` - 获取订单详情
- `GET /api/miniprogram/orders/<order_number>/qrcode` - 生成订单二维码
- `POST /api/miniprogram/orders/<order_id>/upload` - 上传订单图片
- `PUT /api/miniprogram/orders/<order_id>/status` - 更新订单状态
- `PUT /api/miniprogram/orders/<order_id>/images` - 更新订单图片
- `DELETE /api/miniprogram/orders/<order_id>/images/<image_id>` - 删除订单图片

#### 目录相关
- `GET /api/miniprogram/styles` - 获取风格列表
- `GET /api/miniprogram/products` - 获取产品列表

#### 作品相关
- `GET /api/miniprogram/works` - 获取作品列表

#### 推广相关

#### 媒体相关
- `GET /api/miniprogram/media/original/<path>` - 获取原图
- `GET /api/miniprogram/media/final/<path>` - 获取效果图
- `GET /api/miniprogram/media/static/<path>` - 获取静态图片
