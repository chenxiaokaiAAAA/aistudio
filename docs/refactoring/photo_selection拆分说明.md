# photo_selection.py 拆分说明

## 当前状态

由于文件很大（2018行），拆分工作采用渐进式方法：

1. ✅ 已完成基础结构：
   - `utils.py` - 工具函数
   - `list.py` - 订单列表
   - `detail.py` - 订单详情
   - `__init__.py` - 主蓝图注册

2. ⏳ 待完成模块：
   - `submit.py` - 提交选片（435行代码）
   - `confirm.py` - 确认选片（包含4个函数，约400行）
   - `print_module.py` - 打印相关（约300行）
   - `qrcode.py` - 二维码相关（约280行）
   - `search.py` - 搜索相关（约70行）

## 临时方案

在完成所有模块拆分之前，`__init__.py` 会：
1. 注册已完成的子模块（list, detail）
2. 暂时保留原 `photo_selection.py` 文件，确保系统正常运行
3. 待所有模块完成后，再完全切换到新的模块化结构
