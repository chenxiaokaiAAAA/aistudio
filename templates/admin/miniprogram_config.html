小程序基础配置
在微信公众平台 -> 开发 -> 开发设置中获取。若显示admin，可点击「同步」从支付AppID复制
在微信公众平台 -> 开发 -> 开发设置中获取（需要重置后查看）
微信支付配置
通常与小程序AppID相同，或使用支付专用AppID
在微信商户平台 -> 账户中心 -> 商户信息中获取
在微信商户平台 -> 账户中心 -> API安全中设置
支付结果通知的回调URL，需要在微信商户平台配置
提示：修改配置后需要重启服务才能生效。请确保配置信息正确，错误的配置可能导致小程序无法正常使用。
